Создание БД и слоя доступа к данным (DAL).
1. Написать SQL-скрипты создания всех таблиц, первичных и внешних ключей, индексов (по инвентарному номеру).
2. Заполнить таблицы-справочники (Типы оборудования, Подразделения) тестовыми данными (5-10 записей).
3. В Visual Studio создать решение EquipmentAccounting. Добавить проект Class Library DAL.
4. В DAL создать модель классов (Entity: Department, Employee, EquipmentType и т.д.).
5. Реализовать класс DbContext (или аналог на чистом ADO.NET) для подключения к БД. Строку подключения вынести в App.config.
6. Задание повышенной сложности: Реализовать Generic Repository для базовых операций CRUD.
